How-To for using the Yahoo Stock API with PHP

by Kevin To

Background information on the Yahoo Stock API

What is the Yahoo Stock API?

This API lets you get stock information from Yahoo. The information can be retrived through GET calls.

Can I get multiple return data types from this API?

Yes, you can get multiple return types. The return formats are CSV, XML, and JSON. This How-To guide will primarily be for using the CSV file format. The request signature for the CSV file format can be found here. The request signature for the XML and JSON can be found here.

How many times can you call this API?

From my testing, it seems that you can call this API an unlimited number of times. The drawback is that this API does not allow you to get information for all of the stocks at the same time.

How many servers are available for this API?

Currently only the North American server is available to use. This means that if you use this API for a website, you need back up sources for your stock requests. Otherwise, if Yahoo's API goes down, your website will have to operate with old information.

What kind information can you get from this stock API?

You can get stock quotes, historial quotes, and sector/industry/company information. This guide will mainly be focusing on stock quotes.
