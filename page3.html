Section 3: Parsing a CSV file

How to iterate through values of a CSV file

The previous page showed you how to get and save data from the Yahoo API. In this step, you will learn how to iterate through the data contained within a CSV file. The sample code for how to do this:

        
          $row = 1;
          if (($handle = fopen("quotes.csv", "r")) !== FALSE) {
              while (($data = fgetcsv($handle, 1000, ",")) !== FALSE) {
                  $num = count($data);
                  echo "<p> $num fields in line $row: <br/></p>\n";
                  $row++;
                  for ($c=0; $c < $num; $c++) {
                      echo $data[$c] . "<br/>\n";
                  }
              }
              fclose($handle);
          }
